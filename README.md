# Next.js 練習用

# 🛒 ECアプリ（Next.js × Prisma）

商品登録・一覧・詳細表示・カート機能を持つシンプルなECアプリです。  
画像アップロードやカテゴリ機能も実装しています。

---

## 使用技術

- Next.js（App Router）
- TypeScript
- Prisma
- SQLite
- Tailwind CSS
- shadcn/ui

---

## 主な機能

- 商品の作成・一覧・詳細表示
- 商品へのカテゴリ付与
- カート機能
- 合計金額表示
- 画像アップロード（ローカル保存）
- 画像未登録時のフォールバック表示
- カテゴリによる絞り込み

---

## 工夫した点

- Server Actionsを使い、APIを作らずにデータ更新を実装
- Prismaを用いたシンプルで可読性の高いDB操作
- Next/Imageを使った画像表示
- UIをshadcn/uiで整えた
